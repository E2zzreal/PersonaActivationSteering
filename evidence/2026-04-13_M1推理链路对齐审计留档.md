# 2026-04-13 M1 推理链路对齐审计留档

日期：2026-04-13
最后验证日期：2026-04-13
状态：已完成

## 核心发现

1. baseline 与 injected 使用的 prompt 构造方式不同；
2. baseline 显式使用 `/no_think` 与 `enable_thinking=False`，injected 没有等价控制；
3. injected 推理格式与训练 chat-template 格式不一致；
4. injected 使用自定义逐 token 解码，baseline 使用原生 `generate`；
5. 因此当前两者并非严格可比的同口径推理链路。

## 决策含义

- 后续应优先修复“推理链路对齐”；
- 再观察 injected 泄露率是否显著下降；
- 在此之前，不宜把高泄露全部归因到 loss 或注入层本身。
