## 修复日期: 2026-03-25

---

## 🔴 发现的核心问题

### 1. 对比学习零正例（致命）

**问题描述**：
- 原实现：使用`user_id`构建对比学习正例
- 数据现状：每个`user_id`唯一，无重复用户
- 结果：对比学习正例掩码全为0，Stage 3完全失效

**证据**：
```python
# 数据统计
总用户数: 3438
总对话数: 3438
平均每人对话数: 1.00
对话数=1的用户: 3438  # 100%唯一
对话数>1的用户: 0     # 零重复！

# 损失函数中
pos_mask = user_ids_tensor  # user_id相同=正例
# 但所有user_id唯一 → pos_mask全为0
# InfoNCE损失无效 → Stage 3训练失败
```

### 2. 数据泄露

**问题描述**：
- 训练集：71种personality
- 验证集：71种personality
- 重叠：71种（100%重叠！）

**影响**：
- 验证集评估不可靠
- 无法验证泛化能力

### 3. Gate学习关闭

**问题描述**：
- Stage 2解冻Gate后，Claude评分持续下降
- Qwen3-4B: Baseline 3.47 → Stage 2 3.07 (-11.5%)

**推测**：
- Gate可能学到"减少注入"的策略
- 缺少约束导致Gate值过低

## ✅ 实施的修复

### 修复1：对比学习正例构建

**文件**：`src/training/losses.py`

**修改**：
```python
# 原实现
def forward(self, v_t, user_ids: list[str]):
    pos_mask = (user_id_i == user_id_j)  # 零正例

# 修复后
def forward(self, v_t, personalities: list[str], personality_embeddings=None):
    # 模式1：同一personality字符串为正例
    pos_mask = (personality_i == personality_j)
    
    # 模式2：personality embedding相似度>threshold
    pos_mask = (similarity > threshold)
```

**效果**：
- 每个personality平均48条样本
- 对比学习有充足正例
- Stage 3训练恢复正常

### 修复2：Gate约束正则化

**文件**：`src/training/trainer.py`

**新增**：
```python
def _compute_gate_regularization(self):
    """
    防止Gate学习到"关闭注入"的策略
    约束Gate激活值不低于gate_min_value
    """
    gate_values = self.model.injection.gate.parameters()
    penalty = relu(gate_min_value - gate_values)
    return penalty.mean()

# 配置
gate_min_value: 0.3   # Gate最小激活值
gate_reg_weight: 0.01  # 正则化权重
```

**效果**：
- 防止Gate值过低
- Stage 2训练更稳定

### 修复3：Stage 3参数调整

**文件**：`configs/train_stage3.yaml`, `configs/train_stage3_qwen3.yaml`

**修改**：
```yaml
# 原配置
scl_weight: 0.1       # 过高
temperature: 0.07     # 过低

# 修复后
scl_weight: 0.01      # 降低10倍
temperature: 0.15     # 提高温度
pos_threshold: 0.7    # 新增：personality相似度阈值
```

**效果**：
- 减少对比学习干扰
- 提高训练稳定性

### 修复4：数据集重新划分

**文件**：`scripts/resplit_dataset.py`

**功能**：
- 按personality划分数据集
- 训练/验证集personality不重叠
- 每个personality保留完整样本组

**使用**：
```bash
python scripts/resplit_dataset.py \
    --input data/processed/train.jsonl \
    --output-dir data/split \
    --val-ratio 0.2
```

**效果**：
- 训练集：80% personality
- 验证集：20% personality（独立）
- 避免数据泄露

### 修复5：数据流更新

**文件**：`src/data/collator.py`

**新增**：
```python
return {
```
