## 3. 关键结论

### 3.1 旧结论需要修正

此前“baseline 最强、injected 系统性高泄露”的判断，建立在未对齐推理链路的旧口径上。

本轮统一链路重跑后：
- injected 不再表现出 20%~30% 级别的大面积泄露；
- 各 stage 泄露率已降到 `0.00% ~ 2.22%`；
- stage2 以 `3.589` 成为当前小样本口径下的最佳配置。

因此，旧结论应修正为：

> 在统一推理链路后，PersonaSteer injected 模型并不存在此前观察到的系统性高泄露；当前最优配置由 `stage2` 取得，但 stage1 / stage3 与 baseline 的差距都已明显缩小。

### 3.2 当前真正稳定的结论是什么

当前可以稳定成立的结论有：
1. M1 / M1.5 证明推理链路差异曾显著放大 injected 泄露；
2. 统一链路后，泄露已从主矛盾降为次级问题；
3. stage2 当前均分最高；
4. stage1 泄露率最低（0.00%）；
5. stage3 并未在当前口径下继续扩大优势，说明 stage3 训练收益尚不稳定。

### 3.3 为什么 stage2 暂时领先值得重视

这意味着：
- PersonaSteer 并非“越往后阶段越强”；
- 也可能不是“注入越强越好”；
- 当前更可能存在一个较优中间点，使人格化收益和副作用之间达到更好平衡。

## 4. 对计划的影响

后续计划建议调整为：

1. 将“高泄露止损”从主线降级；
2. 将“统一链路下的 stage 对比与稳定性验证”提升为主线；
3. 优先验证：
   - stage2 是否在更大样本上仍领先；
   - stage1 / stage3 是否只是统计波动；
   - 是否需要在 stage2 基础上再做小规模层位优化。

## 5. 一句话结论

> 当前权威结论已经改变：在统一推理链路后，PersonaSteer 的 injected 模型不再表现出此前那种系统性 thinking 泄露；在 90 条样本的小规模权威重跑中，`stage2` 以 `3.589 / 2.22%` 成为当前最佳配置，baseline 不再是唯一最优参考。
